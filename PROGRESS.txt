ClawOne 项目状态报告
===================

项目概述
--------
ClawOne 是一个基于 Electron + React + TypeScript 的 OpenClaw 一键安装器。
当前版本：1.0.3
最近提交：feat: add auto-update functionality

技术栈
------
- Electron 28+
- React 18+
- TypeScript 5+
- Vite 5
- electron-builder
- electron-updater (自动更新)
- i18next + react-i18next (国际化)

项目结构
--------
/electron
  - main.ts: Electron 主进程，处理 IPC 通信和系统调用
  - preload.ts: 预加载脚本，暴露安全的 API 给渲染进程

/src
  - App.tsx: 主应用组件，管理页面路由和配置状态
  - types.ts: TypeScript 类型定义
  - App.css: 全局样式
  - main.tsx: React 入口文件

/src/i18n
  - index.ts: i18n 配置和初始化
  - en.json: 英文翻译
  - zh.json: 中文翻译

/src/pages
  - Welcome.tsx: 欢迎页面
  - Environment.tsx: 环境检测页面（Node.js ≥22, npm, Git）
  - Install.tsx: OpenClaw 安装页面
  - ConfigStep1.tsx: 配置步骤1 - AI Provider 选择
  - ConfigStep2.tsx: 配置步骤2 - API Key 输入
  - ConfigStep3.tsx: 配置步骤3 - Workspace 配置
  - ConfigStep4.tsx: 配置步骤4 - Channels 选择
  - Dashboard.tsx: 服务管理仪表板

核心功能
--------
1. 环境检测
   - 检测 Node.js 版本（要求 ≥22）
   - 检测 npm 安装状态
   - 检测 Git 安装状态

2. 一键安装
   - 执行 npm install -g openclaw@latest
   - 实时显示安装日志

3. 配置向导（4步）
   - AI Provider 选择（Anthropic, OpenAI, Ollama 等）
   - API Key 配置和连接测试
   - Workspace 目录配置
   - 消息通道选择（WhatsApp, Telegram, Discord, Slack）

4. 服务管理
   - 启动/停止 OpenClaw 服务
   - 实时服务状态监控
   - 服务日志查看
   - 配置信息展示

5. 国际化支持
   - 中英文双语界面
   - 自动检测系统语言
   - 手动切换语言（右上角按钮，带动画效果）
   - 语言偏好持久化存储
   - 所有文本完全翻译，无硬编码
   - 专业术语准确翻译

6. 自动更新
   - 启动时自动检查更新（仅生产环境）
   - 手动检查更新按钮
   - 更新可用通知
   - 下载进度显示
   - 用户控制下载和安装时机
   - GitHub Releases 集成
   - 跨平台更新支持

IPC 通信接口
-----------
- check-node: 检查 Node.js 版本
- check-package-manager: 检查包管理器
- check-git: 检查 Git
- install-openclaw: 安装 OpenClaw
- read-config: 读取配置文件 (~/.openclaw/openclaw.json)
- write-config: 写入配置文件
- test-api-connection: 测试 API 连接
- start-openclaw: 启动服务
- stop-openclaw: 停止服务
- get-service-status: 获取服务状态
- check-for-updates: 检查更新
- download-update: 下载更新
- install-update: 安装更新并重启
- write-config: 写入配置文件
- test-api-connection: 测试 API 连接
- start-openclaw: 启动服务
- stop-openclaw: 停止服务

跨平台支持
----------
✅ Windows 10+ (64-bit)
   - NSIS 安装器
   - 便携版 (portable)
   - 使用 cmd.exe 执行命令
   - 标准窗口标题栏

✅ macOS 10.13+ (Intel & Apple Silicon)
   - DMG 安装包
   - ZIP 压缩包
   - hiddenInset 标题栏样式 + frame: true（支持窗口拖动）

✅ Linux (Ubuntu 18.04+, Debian 10+, Fedora 32+)
   - AppImage
   - DEB 包

跨平台实现细节
--------------
- execCommand(): 跨平台命令执行（自动处理 Windows shell）
- spawnCommand(): 跨平台进程生成（Windows 使用 cmd.exe）
- 平台检测：process.platform === 'win32'
- 路径处理：使用 path.join() 和 os.homedir()
- 标题栏样式：macOS 使用 hiddenInset，其他平台使用默认

构建配置
--------
- 开发模式: npm run electron:dev
- 生产构建: npm run electron:build
- 输出目录: release/
- 支持平台: macOS (dmg/zip), Windows (nsis/portable), Linux (AppImage/deb)
